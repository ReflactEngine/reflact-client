# Reflact Client

A Fabric mod for Minecraft 1.21.11 designed to interface with the Reflact Server. It handles custom rendering, inputs, and client-side logic.

## Features

*   **HUD:** Custom `ReflactHud` for mana and status display.
*   **Tooltips:** `RpgTooltipRenderer` for custom item stats.
*   **Input:** Keybindings (e.g., 'R' to cast spells).
*   **Networking:** Syncs state with the server via CustomPayloads.

## Running

Launch the Minecraft client:

```bash
./gradlew runClient
```
